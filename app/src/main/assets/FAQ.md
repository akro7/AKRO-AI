# IRazor AI — FAQ

---

**1. What is IRazor AI?**

IRazor AI is an Android app built by the one dev Ali Alsaadi. Ai It gives you a powerful AI coding assistant that can build apps games websites and write professional code — all from your phone or website pc

---

**2. What model does it use?**
IRazor AI runs on the **IQD1R model**, developed and powered through the OpenCode infrastructure.

---

**3. Is it like DeepSeek? Is it fully free?**
Yes — same idea. You get full AI power with no subscription and no Limits Use / IRazor AI comes with built-in free API keys that rotate automatically, so it works out of the box and You can also add your own key in Settings if you want.

---

**4. What can it do?**
- Generate complete Android apps, websites, and games
- Read, edit, and write files directly on your device
- Search the web
- Understand your full project folder (just paste the path)

---

**5. Does it work offline?**
No — it needs an internet connection to call the AI api.

---

**6. How do I use my own API key?**
Go to **Settings → Custom API Key** and paste your key. It must start with `sk-`. The app will use it instead of the built-in keys automatically.

---

**7. How do I connect GitHub (beta 100%) need fix or update**
Go to **Settings → GitHub Token**, generate a Classic token from github.com/settings/tokens with the `repo` scope checked, and paste it. Then you can say things like *"push this file to my repo"* and it will do it.

---

**8. What file types can it read?**
Any text file up to **25MB** — `.kt`, `.java`, `.xml`, `.gradle`, `.json`, `.html`, `.js`, `.py`, and more. Binary files (images, APKs, ZIPs) are skipped automatically.
